# 消费账单分类分析系统

## 项目简介

多平台消费账单（支付宝/微信/抖音）的统一解析、自动分类、统计分析工具。支持多用户（家庭成员），自定义分类体系，Web 端展示，移动端适配。

## 项目结构

```
消费账单记录/
├── 支付记录样例/          # 三个平台的账单样例文件
│   ├── 支付宝交易明细(20260101-20260531).csv
│   ├── 微信支付账单流水文件(20260101-20260531)_20260619221302.xlsx
│   └── dy_202606192215440510082987796911260.pdf
├── Docs/                  # 项目文档
│   ├── 需求说明.md        # 需求说明
│   ├── 技术方案.md        # 数据库设计、API设计、部署架构
│   └── 执行计划.md        # 开发步骤
├── 开发日志/              # 按日期记录开发进展
├── backend/               # Python FastAPI 后端
└── frontend/              # SvelteKit 前端
```
