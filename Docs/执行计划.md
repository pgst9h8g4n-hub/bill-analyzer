## 阶段三：分类系统与API

### 3.1 分类管理 API
- [ ] GET /api/categories — 获取分类树
- [ ] POST /api/categories — 创建分类
- [ ] PUT /api/categories/:id — 更新分类
- [ ] DELETE /api/categories/:id — 删除分类
- [ ] POST /api/categories/import-template — 导入模板

### 3.2 关键词自动分类引擎
- [ ] 实现 classify_record() 函数
- [ ] 多级评分匹配（商户完全匹配1.0 > 包含0.8 > 商品描述0.6 > 全文0.4）
- [ ] 阈值控制（低于0.5标记为"未分类"）
- [ ] 支付宝自带分类映射
- [ ] 批量分类接口

### 3.3 账单上传与解析 API
- [ ] POST /api/upload — 接收文件，存入 Supabase Storage
- [ ] POST /api/upload/:id/parse — 触发解析
- [ ] GET /api/upload/:id/status — 查询解析进度
- [ ] 去重逻辑（基于 platform_order_no）

### 3.4 账单记录 API
- [ ] GET /api/records — 分页查询（支持多筛选条件）
- [ ] PUT /api/records/:id/category — 手动修正分类
- [ ] POST /api/records/batch-update — 批量修正
- [ ] GET /api/records/export — 导出 CSV
